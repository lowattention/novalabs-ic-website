# novalabs-ic-website
Static version of the Nova Labs Innovation Center Website

## Local Development

**Prerequisites:** Node.js

**Install dependencies** (first time only):
```sh
npm install
```

**Start the dev server:**
```sh
node serve.mjs
```

The site is served at [http://localhost:3000](http://localhost:3000).

The server serves static files from the project root. Edit `index.html` and refresh the browser to see changes.
